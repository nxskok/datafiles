Jan 25 lunchtime: people have been asking about copying and pasting a SAS graph into a word doc. Marta hit upon the idea of using "Print Screen", which probably works just fine. I also found this by googling, and it works for me:

  • In the SAS graph window, select Edit and Copy (even though nothing is selected). Wait. Eventually SAS will report that the graph was copied to the clipboard.
  • Go to Word, and to the place where you want the plot to appear. Select Edit and Paste Special. One of the options for what the thing is you're trying to paste should be Device-Independent Bitmap. Select that, and then OK. (Strictly, this worked for me in WordPad, since I don't have Word.)
Anything that ends up with a printed copy of the graph in the assignment is fine by me.

I also found out how you can select just some of the output window. Click on where you'd like to start copying. Select Edit, and Select (if you see what I mean). Click on the end of where you want to copy. The whole thing should get highlighted. Then select Edit and Copy, and paste as normal.
